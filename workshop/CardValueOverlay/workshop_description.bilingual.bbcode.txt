[h1]Card Value Overlay[/h1][i]English / 简体中文[/i][h2]Overview[/h2]Card Value Overlay simulates how one card changes the expected value of the current deck. It currently focuses mainly on Regent and Colorless cards.[h2]Requirements and usage[/h2][list][*]Requires BaseLib.[*]Displays one real-time dEV table above cards.[*]Simulation parameters can be adjusted in the BaseLib mod configuration.[*]Client-side display only; multiplayer teammates do not need the mod.[/list][h2]Display guide[/h2][list][*][b]short / mid / long[/b]: 4 / 8 / 12-turn horizons.[*][b]add dEV[/b]: deck value after adding the offered card minus current deck value.[*][b]owned dEV[/b]: current deck value minus deck value after removing this one copy.[*][b]upgrade dEV[/b]: deck value after replacing the unupgraded copy with the upgraded copy minus current deck value.[*][b]mean / 95% CI[/b]: paired per-run mean and paired Student-t confidence interval.[/list]Sampling advances through 15, 30, 45, and 60 runs. Simple cards may stop when all three horizons have a stable sign; complex cards use at least 30 runs. Early stopping uses a stricter four-look Bonferroni interval. Green/red appears only when that interval is wholly positive/negative. Suffixes: ~ means preview/refining, ok means stable, and ? means the maximum was reached without all horizons stabilizing. dEV is local and horizon-limited; unsupported or complex effects may still be quantified imperfectly.[h2]Plans[/h2][list][*]Support relics.[*]Add other characters.[*]Improve simulator policy and card-effect coverage.[/list][h2]Credits[/h2]Inspired by Solisora's card-value framework; the modeling and implementation are my own. Feedback is welcome.[hr][/hr][h1]Card Value Overlay 中文说明[/h1][h2]简介[/h2]本 Mod 会实时模拟某一张牌使当前卡组的期望价值发生多少变化，目前主要支持储君与无色卡。[h2]前置与使用[/h2][list][*]前置需求：BaseLib。[*]在卡牌上方只显示一张实时 dEV 表。[*]可在 BaseLib 的 Mod 配置中调整模拟参数。[*]仅在本地显示信息；联机队友不需要安装。[/list][h2]显示内容[/h2][list][*][b]short / mid / long[/b]：4 / 8 / 12 回合视角。[*][b]add dEV[/b]：加入候选牌后的卡组价值减去当前卡组价值。[*][b]owned dEV[/b]：当前卡组价值减去移除这一张牌后的价值。[*][b]upgrade dEV[/b]：用升级形态替换未升级牌后的卡组价值减去当前卡组价值。[*][b]mean / 95% CI[/b]：逐轮配对差值的平均数与配对 Student-t 置信区间。[/list]模拟按 15、30、45、60 轮逐批执行。简单牌可在三个时间线的符号都稳定后提前停止；复杂牌至少计算 30 轮。提前停止使用更严格的四次查看 Bonferroni 区间，只有该区间完全为正或负时才显示绿色或红色。后缀中，~ 表示预览或继续加算，ok 表示稳定，? 表示达到最大轮数后仍未全部稳定。dEV 是局部且受时间线限制的指标，复杂或未支持的效果仍可能量化不准。[h2]计划[/h2][list][*]支持遗物。[*]追加其他职业。[*]改善模拟器策略与卡牌效果覆盖。[/list][h2]致谢[/h2]思路受到 Solisora 价值化卡牌体系启发，具体建模与实现由我独立完成。欢迎提出建议。
